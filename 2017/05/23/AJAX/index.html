AJAX详解

发表于 2017-05-23   |   分类于 JavaScript

  AJAX是Asynchronous JavaScript and XML的缩写。它能够只刷新局部网页的数据，而不是重新加载整个页面。

Ajax请求的五个步骤

建立XMLHttpRequest对象

注册回调函数

使用open()和send()方法发送资源请求给服务器

向服务器端发送数据

在回调函数中判断服务器是否正确返回数据

Ajax请求总共有8种callback

onSuccess
onFailure
onUninitialized
onLoading
onLoaded
onInteractive
onComplete
onException
